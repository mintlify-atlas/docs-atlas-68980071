## Why BetterWinTab?

Modern power users run dozens of windows simultaneously — multiple VS Code instances, browser profiles, terminals, and dashboards — yet the built-in `Alt + Tab` still shows a flat, recency-ordered list with no context. BetterWinTab fixes this with three core ideas:

- **Context** — windows are organized into folders by app, project, or any custom rule you define.
- **Search** — type any part of a title, process name, or app name to instantly filter every open window.
- **Speed** — the overlay opens, filters, and closes without ever requiring the mouse.

## Get Up and Running

<Steps>
  <Step title="Download BetterWinTab">
    Grab the latest installer from the [GitHub Releases page](https://github.com/sgm1018/BetterWinTab/releases). Run `BetterWinTab-Setup-x.x.x-x64.exe` and follow the wizard.
  </Step>
  <Step title="Launch the overlay">
    After installation, BetterWinTab starts in the system tray. Press `Ctrl + Tab` to open the window switcher overlay.
  </Step>
  <Step title="Navigate and focus">
    Use `Arrow keys` to browse windows, `Tab` / `Shift + Tab` to switch folders, and `Enter` to focus the selected window. Start typing at any time to search.
  </Step>
  <Step title="Customize">
    Open **Settings** from the overlay or system tray to configure your hotkey, create Smart Folders, and pick a theme.
  </Step>
</Steps>

<Note>
  BetterWinTab requires Windows 10 (build 19041+) or Windows 11. It is a native WinUI 3 application — no Electron, no web runtime, no admin rights required for the default per-user install.
</Note>
